# 函数
## 函数特点
- 函数是完成特定任务的独立程序代码单元。

- 使用函数可以省去编写重复代码的苦差，让程序更加模块化，提高了程序代码的可读性，方便后期修改、完善。

- 函数原型可以置于main前面也可以放在main里面的声明变量处。

- 局部变量与其他地方的变量可以重名。
